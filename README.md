**# 🍕 Pizza Sales Performance Dashboard – Power BI**

## 📌 Project Overview
This project is an interactive **Pizza Sales Performance Dashboard** developed using **Power BI** to analyze sales performance, revenue trends, customer ordering behavior, and product insights.

The dashboard transforms raw pizza sales data into meaningful business insights using **Power BI, SQL, DAX, and Data Visualization techniques**.

## 🎯 Project Objectives
- Analyze overall sales performance
- Track total orders and revenue
- Identify best and worst performing pizzas
- Monitor monthly sales trends
- Compare pizza categories and sizes
- Generate business insights through dashboards
  
## 🛠 Tools & Technologies
- Power BI
- SQL
- DAX
- Power Query
- Excel / CSV
- Data Visualization

## 📂 Dataset

The dataset includes:
- Order Details
- Revenue Information
- Pizza Categories
- Pizza Sizes
- Sales Metrics
- Customer Ordering Data
